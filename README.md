## Project Overview

This project presents an interactive Excel dashboard designed to analyze global cybersecurity threats from 2015 to 2024. The dashboard visualizes cyber attack trends across multiple countries and industries, helping identify patterns in attack types, financial losses, affected users, and security vulnerabilities.

## Objectives

* Analyze cybersecurity threats across different countries and industries.
* Understand the financial impact caused by cyber attacks.
* Identify the most common attack types and vulnerabilities.
* Visualize cybersecurity data using an interactive dashboard.

## Dashboard Features

* Key Performance Indicators (KPIs):

  * Total Cyber Attacks
  * Total Financial Loss
  * Total Users Affected
  * Average Incident Resolution Time
  * Total Countries Analyzed

* Data Visualizations:

  * Top Cyber Attack Types
  * Attack Source by Industry
  * Financial Loss by Country
  * Users Affected by Country
  * Security Vulnerability Analysis
  * Defense Mechanisms Used

* Interactive Filters:

  * Year
  * Country

## Tools Used

* Microsoft Excel
* Pivot Tables
* Data Visualization
* Dashboard Design

## Dataset Information

The dataset includes information on:

* Country
* Year
* Attack Type
* Target Industry
* Financial Loss (Million $)
* Number of Affected Users
* Attack Source
* Security Vulnerability Type
* Defense Mechanism Used
* Incident Resolution Time (Hours)
